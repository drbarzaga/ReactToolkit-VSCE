## ✨ Features

- **📚 Resource Library** — 239+ hand-picked resources from official docs to cutting-edge libraries, always up to date.
- **🔍 Instant Search** — Filter any resource or category in real time from the search bar.
- **📂 37 Categories** — Everything neatly organized and collapsible for distraction-free browsing.
- **🖼️ Visual Logos** — Every resource displays its logo so you can spot what you need at a glance.
- **🌓 Theme Compatible** — Adapts seamlessly to VS Code light and dark themes.


## 🚀 Getting Started

1. Install **ReactToolkit** from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=drbarzaga.reacttoolkit).
2. Click the ReactToolkit icon in the activity bar to open the sidebar.
3. Browse categories or type in the search bar to find what you need.
4. Click any resource to open its website or documentation.

> You can also open the sidebar with `Ctrl+Shift+P` / `Cmd+Shift+P` → **ReactToolkit: Show Resources**.
